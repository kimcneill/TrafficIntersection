# TrafficIntersection
Inefficient traffic light timing at intersections causes unnecessary
congestion, leading to increased vehicle wait times, longer maximum
queue lengths, and reduced overall intersection throughput. This
inefficiency contributes to broader real-world issues, including
increased travel time, higher fuel consumption, and greater pollution. ​

The goal of this project is to create an event-driven simulation of a four-way traffic intersection and
compare different traffic light timing strategies to reduce average delay, limit queue growth, and
improve throughput.​
